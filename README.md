# Amazon Sales Dashboard

# Overview
* This project is an interactive sales analytics dashboard built in Microsoft Excel using Power Query, Pivot Tables, Pivot Charts, and Slicers.
* The dashboard provides a comprehensive view of sales performance across categories, sub-categories, states, months, and payment methods. It enables users to explore trends, monitor key business metrics, and gain actionable insights from sales data.

# Project Objectives
* Clean and transform raw sales data using Power Query.
* Create an interactive dashboard for business reporting.
* Track important sales KPIs.
* Analyze customer purchasing behavior.
* Identify sales trends across products and regions.
